This capstone project is a full-stack dating platform built to explore intelligent discovery through scalable backend design and AI-driven recommendation logic.

The system is architected using FastAPI with a modular service layer, SQLAlchemy for ORM abstraction, and PostgreSQL in a production-ready schema. It exposes structured RESTful APIs to support onboarding, profile management, match computation, and messaging workflows. Authentication is implemented using JWT-based access control with protected routes and validated request pipelines.

The discovery engine combines preference-based filtering with compatibility scoring to generate personalized match rankings. The architecture is designed to evolve toward embedding-based similarity search and adaptive ranking strategies, enabling experimentation with recommendation system techniques.

The frontend is built in React with state-driven rendering to support dynamic feed updates and interaction flows. Media handling is integrated via Cloudinary for scalable storage and transformation, and the backend is structured for deployment on AWS infrastructure.

The project emphasizes clean API design, normalized relational modeling, modular backend architecture, and scalability considerations. It reflects an approach to building user-facing systems that balances technical rigor with thoughtful product design.
