## Interactive Analytics Platform

**Frontend:** React + D3.js dashboards with multi-dimensional filtering (categories, regions, date ranges). Real-time chart updates, custom tooltips, responsive SVG visualizations.

**Backend:** Python pipeline—Pandas for data transformation, statistical analysis (z-tests for regional differences, correlation for seasonality). NumPy for vectorized computation on 50K+ transactions.

**Insights:** 40% Q4 sales lift (holiday spike), 20% summer decline, 3x higher LTV for repeat customers. Regional price elasticity varies by segment.

**Why it matters:** Demonstrates full-stack data work—collection, transformation, statistical analysis, and visualization that drives decisions.
